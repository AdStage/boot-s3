# boot-s3

[Boot](http://boot-clj.com/) task to sync local directory with AWS S3.
Most of the code borrowed from [lein-s3-sync](https://github.com/kanej/lein-s3-sync).

## Install

```
[hashobject/boot-s3 "0.1.0-SNAPSHOT"]
```
